passGen

A simple and beginner-friendly desktop **passGen** built with Python and Tkinter. This GUI tool enables users to generate strong passwords, save them locally, and retrieve them when needed â€” all within a single, lightweight app.


---

## Features

- Generate secure random passwords
- Save passwords with usernames ( TO BYPASS USERNAME YOU MAY ENTER A 'SPACE' OR MEANINGLESS SYMBOLS)
- Retrieve stored usernames and passwords
- Simple and clean graphical interface
- Works with or without a username (see usage)

---

## Requirements

- Python 3.x
- Tkinter (usually comes pre-installed)
